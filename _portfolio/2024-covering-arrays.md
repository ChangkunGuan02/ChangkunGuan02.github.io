Covering arrays are a core tool in **combinatorial interaction testing**: compact test suites that guarantee coverage of all parameter-value interactions up to a given strength. This project studied how to optimize covering arrays with binary parameters under constraints.

**Key contributions:**
- Worked on proofs about the structure of optimal covering arrays under constraints
- Implemented integer programming models in Gurobi and algorithms in Python
- Proposed the project and received funding from Bucknell's Program for Undergraduate Research

**Principal Investigator:** Dr. Thiago Serra
